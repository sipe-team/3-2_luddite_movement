# Luddite Movement

- 마우스를 사용하지 않고 개발과 업무를 하는 것을 목표로 하는 미션이에요.
- 이를 위해 다양한 마우스 사용, 반복적인 업무를 자동화하는 것을 목표로 해요.

## 미션 일정

- 0주차 (~12/22): 목표와 계획 세우기 (1)
- 1주차 (~12/29): ing
- 2주차 (~01/05): ing
- 3주차 (~01/12): 1차 회고, 목표와 계획 세우기 (2)
- 4주차 (~01/19): ing
- 5주차 (~01/26): ing
- 6주차 (~01/31): 2차 회고, 마무리, 2차 미션 발표 준비
- 2차 미션 발표 (02/01)
